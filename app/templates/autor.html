Sobre o Projeto

A Base Mestra SEI é um sistema de enriquecimento automático de dados fiscais desenvolvido no contexto do Workshop CeMEAI. O projeto aplica modelos de linguagem de grande escala (LLMs) para normalizar descrições de produtos, inferir NCMs corretos e calcular scores de confiança sobre dados de Notas Fiscais Eletrônicas (NF-e) do setor público.

O pipeline processa GTINs com prefixo 789 (GS1 Brasil), aplica clustering semântico com DBSCAN e extrai atributos estruturados via LLM (Gemma local ou GPT-4o-mini), gerando uma base mestra auditável com rastreabilidade completa.
