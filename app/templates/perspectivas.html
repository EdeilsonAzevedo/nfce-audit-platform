A Base Mestra SEI é uma fundação para um ecossistema maior de inteligência fiscal e econômica. As iniciativas abaixo representam extensões naturais do pipeline atual, organizadas por impacto e esforço.

NER Customizado

Baixa prioridade

Usar o LLM para anotar ~500–1000 descrições e treinar um modelo leve de NER (spaCy ou BERTimbau) que roda localmente em escala. Após o treino, a extração é instantânea e gratuita — sem depender de API.

PRODUTO MARCA PESO VARIANTE UNIDADE

Dashboard de Preços

Média prioridade

Interface interativa conectada à base padronizada para consultar preço médio por município e período, comparar entre regiões via mapa e emitir alertas de variação anômala.

Analistas SEI Jornalistas Consumidores

Detecção de Anomalias

Média prioridade

Modelo estatístico ou ML para sinalizar preços fora do padrão: outliers dentro do mesmo cluster, variação temporal suspeita (fraude fiscal ou erro de cadastro) e insumo direto para auditoria da SEFAZ.

Matching Geoespacial

Exploratória

Identificar se o mesmo produto tem preços sistematicamente diferentes entre regiões. Mapa de calor por município, ranking de municípios mais caros por categoria e detecção de isolamento logístico nos preços.

Bases Externas

Exploratória

Integração com referências externas para enriquecimento e validação cruzada do dicionário gerado.

Open Food Facts — validação cruzada do dicionário GTIN
Tabela TACO — enriquecimento nutricional dos alimentos
IBGE (POF, IPCA) — contextualização frente a índices nacionais

API REST do Dicionário

Baixa prioridade

API FastAPI leve que expõe o dicionário padronizado para outros sistemas da SEI e SEFAZ. Pode virar um serviço permanente alimentado por novos lotes de NFC-e.

GET /produto?gtin=7891000100103

Pipeline Incremental

Baixa prioridade

O pipeline atual trata um snapshot. Dados de NFC-e chegam continuamente — novos lotes devem verificar se o GTIN já existe, processar apenas o que é novo e monitorar drift de descrições que não se encaixam nos clusters existentes.

Generalização para Outros Estados

Exploratória

O pipeline é agnóstico ao estado — com pequenos ajustes pode ser aplicado em dados de NFC-e de SP, MG, RS e outros. Vocabulários de abreviações regionais diferentes e potencial de parceria com outros SEIs estaduais.
